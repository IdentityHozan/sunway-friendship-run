Event Details

Event Overview

The Sunway Pre-U Friendship Run is a non-competitive charity run designed for students, alumni, friends, and community members.

Participants may walk, jog, run-walk, or run the route at their own pace. The focus is not only fitness, but friendship, stress relief, and community.
